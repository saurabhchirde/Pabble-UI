Introduction

Pebble UI is an open source styling framework, which helps you to quickly build your Dream Project, by focusing much on Functionality, because we take care of your Designs.

Installation

To use Pebble UI components in your project, Copy and paste the below CSS link in the '<head>' tag of your html, above all other style sheets.

Font

For all of our components we have used "Inter" is default font. Still you can use another font of your choice, by placing the new font styling below our CDN in your '<head>' tag.

CDN for "Roboto" font is given below. If you want to use it, just paste it in your '<head>' tag.
